# Python para Data Science: Funções, Estruturas de Dados e Exceções

Este repositório contém os códigos desenvolvidos durante o curso **Python para Data Science: trabalhando com funções, estruturas de dados e exceções**. O objetivo deste curso foi aprofundar os conhecimentos sobre Python e suas aplicações em Data Science, abordando tópicos como funções, estruturas de dados compostas, list comprehension e tratamento de exceções.

## Tecnologias Usadas
- **Linguagem**: Python
- **Ambiente de Desenvolvimento**: Google Colab
- **Bibliotecas**: Nenhuma biblioteca externa foi utilizada, apenas funcionalidades nativas do Python.

## Conceitos Abordados

Durante o curso, foram abordados diversos conceitos de Python, principalmente no contexto de Data Science. Entre os principais tópicos, destacam-se:

- **Funções e Pacotes**: Entendimento sobre como as funções e pacotes nativos do Python funcionam.
- **Manipulação de Built-in Functions**: Utilização das funções embutidas no Python e suas utilidades em tarefas cotidianas.
- **Funções Personalizadas**: Criação de funções para resolver problemas específicos, utilizando parâmetros, retorno de valores e escopo de variáveis.
- **Estruturas de Dados Compostas**: Manipulação de listas, dicionários e outras estruturas de dados, com ênfase em dados aninhados.
- **List e Dict Comprehension**: Construção de listas e dicionários de forma eficiente e concisa.
- **Tratamento de Erros e Exceções**: Como identificar e tratar diferentes tipos de erros e exceções, garantindo que o código seja robusto e evite falhas inesperadas.

## Exemplos de Códigos

O repositório contém notebooks que demonstram a aplicação dos conceitos acima, bem como resoluções de exercícios utilizando os conhecimentos adquiridos.
